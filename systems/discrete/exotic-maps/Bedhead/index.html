    Bedhead¶

    Systems · Discrete · Exotic maps

    Bedhead attractor -- a sinusoidal planar strange-attractor map.

    Dimension: 2

    Equations¶

    @staticmethod
def _step(X, a, b):
    x, y = X
    xp = np.sin(x * y / b) * y + np.cos(a * x - y)
    yp = x + np.sin(y) / b
    return xp, yp

    Parameters¶

    parameter default
    a -0.67
    b 0.83

    Usage¶

    import tsdynamics as ts

sys = ts.systems.Bedhead()
traj = sys.iterate(steps=10_000)
