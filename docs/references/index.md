# References

Reference material for TSDynamics: how it compares to other libraries, the
literature behind its methods and systems, and the complete API.

This section will cover:

- **Benchmarks** — timing and accuracy comparisons against other dynamical-
  systems libraries.
- **Bibliography** — the original papers behind the methods and the built-in
  [systems](../systems/).
- **API reference** — the full public API, generated from the source
  docstrings: the family base classes, derived wrappers, the
  [analysis toolkit](../analysis/), the [visualization layer](../visualization/),
  state-space primitives, solvers, the registry, and utilities.
