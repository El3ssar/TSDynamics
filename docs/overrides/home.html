Dynamical systems & chaos analysis · Python on a Rust engine

Define the math once, as one symbolic method. TSDynamics lowers it to a native engine and hands you trajectories, Lyapunov spectra, bifurcation diagrams, fractal dimensions, recurrence, basins and more — across 149 built-in ODEs, delay equations and maps, with no compile step and no warmup.

From equations to a chaotic attractor

Instantiate a system, integrate, read named components, and measure the Lyapunov spectrum — the right-hand side is lowered to the Rust engine in-process, so the first call runs at full speed. 

import tsdynamics as ts

lor = ts.systems.Lorenz()
traj = lor.integrate(final_time=100.0, dt=0.01)

traj["x"]                 # named component access
traj.after(20.0)            # drop the transient

lor.lyapunov_spectrum()    # ≈ [0.906, 0.0, -14.57]
ts.kaplan_yorke_dimension(_)  # ≈ 2.06
